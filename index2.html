Python

Python is een op hoog niveau geïnterpreteerde programmeertaal waarvan de filosofie de leesbaarheid van de code benadrukt en wordt gebruikt om allerlei soorten applicaties te ontwikkelen, bijvoorbeeld: Instagram, Netflix, Spotify, Panda3D, onder andere.

JavaScript

JavaScript is een geïnterpreteerde programmeertaal, een dialect van de ECMAScript-standaard. Het wordt gedefinieerd als objectgeoriënteerd, op prototypen gebaseerd, imperatief, zwak getypeerd en dynamisch.

C#

C# is een programmeertaal met meerdere paradigma's, ontwikkeld en gestandaardiseerd door het bedrijf Microsoft als onderdeel van zijn .NET-platform, dat later door ECMA en ISO als standaard werd goedgekeurd. C# is een van de programmeertalen die is ontworpen voor een gemeenschappelijke taalinfrastructuur.

Ruby

Ruby is een geïnterpreteerde, reflecterende, objectgeoriënteerde programmeertaal gemaakt door de Japanse programmeur Yukihiro "Matz" Matsumoto, die in 1993 aan Ruby begon te werken en deze in 1995 publiekelijk introduceerde.
